# [Linux]指令 

## 建立目錄並且切換到新的目錄:

```
$ mkdir /home/foo/test && cd $_
```

## echo 新增檔案內容

``` 
$ echo "hello" > test.md
$ echo "go" >> test.md
```

輸出

```
$ more test.md
hello
go
```
